## 🆘 Troubleshooting

**Problem:** Colors look different
→ Check CSS files are imported in order

**Problem:** Focus outline not visible
→ Press Tab, check browser supports :focus-visible

**Problem:** Buttons don't look right
→ Verify both `.button` and `.button--primary` classes

**Problem:** Spacing is off
→ Use multiples of 8px, use CSS variables

**Problem:** Accessibility failing
→ Use [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
