# PayPal Subscription Process (Customer-Facing)

This document describes the customer-facing subscription flow for ChamberOfCommerceAI using PayPal.

## Overview

Chambers can subscribe to the hosted (SaaS) plan via PayPal. The self-hosted tier remains free.

## Customer Flow

1) **Select Plan**
   - The Chamber chooses a paid plan from the pricing page or email invitation.

2) **Checkout via PayPal**
   - The user is redirected to PayPal to complete payment.
   - PayPal confirms the subscription and returns the user to ChamberOfCommerceAI.

3) **Account Activation**
   - The Chamber receives an activation confirmation email.
   - Access is granted immediately upon payment confirmation.

4) **Manage Subscription**
   - Customers can update payment methods, cancel, or upgrade from the PayPal billing portal.
   - Changes take effect at the next billing cycle unless otherwise stated.

## What Customers Receive

- Hosted access to the ChamberOfCommerceAI platform
- Ongoing updates and security patches
- Priority onboarding support for Chambers
